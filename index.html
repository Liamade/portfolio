Projects

Capstone · Featured Project

ChirpLabs

A fully automated student lab provisioning platform designed and built as my senior capstone. ChirpLabs orchestrates the entire lifecycle of virtualized lab environments — from base image creation to live student access — with zero manual intervention. Terraform provisions VM infrastructure on Proxmox; Packer bakes hardened base images; Ansible handles post-provisioning configuration; Forgejo CI/CD triggers pipelines on commit; Docker containerizes supporting services; PostgreSQL tracks state; and Active Directory manages identity.

Terraform Packer Ansible Proxmox Forgejo CI/CD Docker PostgreSQL Active Directory
Architecture Highlights
  • Full IaC pipeline from infrastructure definition to live environment
  • Git-triggered CI/CD with automated provisioning on push
  • Hardened VM images via Packer with repeatable build specs
  • PostgreSQL-backed state tracking for lab lifecycle management
  • AD-integrated identity with role-based student access
  • Fully self-hosted stack — no cloud dependency
